Установка

1. Скачиваем node.js с интернета
2. в терминале вводим следующие команды:

- npm i
- npm install -g gulp
- npm install --save-dev gulp gulp-sass gulp-uglify gulp-cli

2. Включить live server
3. Вводим npm run start
